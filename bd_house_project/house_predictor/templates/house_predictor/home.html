About Our Project

This Django-based web application predicts house rent prices in Bangladesh using machine learning algorithms trained on local market data.

Data Sources

Our model is trained on comprehensive property listing data from across Bangladesh, including:

  • Property listings from major cities
  • Historical rental price data
  • Property features and amenities
  • Location-based pricing factors

Technology Stack

  • Django web framework
  • Machine learning with Scikit-Learn
  • Python for backend processing
  • SQLite database
  • HTML/CSS/JavaScript for frontend
